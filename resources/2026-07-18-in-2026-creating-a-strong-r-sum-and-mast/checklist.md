# Application Proof Checklist (one per role)

Copy this for each job you apply to.

## Role
- Company: __________
- Title: __________
- Date applied: __________

## Résumé pass
- [ ] Single-column, ATS-safe formatting
- [ ] Copy-paste test into plain text = clean output
- [ ] Keywords from JD present (see prompt #1)
- [ ] Every bullet has a metric OR a concrete outcome
- [ ] No claim I couldn't defend in a follow-up question
- [ ] Removed unnecessary age/date signals unless required

## Proof layer
- [ ] At least one verifiable artifact linked/attached
- [ ] Portfolio link works and is up to date
- [ ] Each headline claim maps to a proof (see prompt #3)

## Interview prep
- [ ] 5 STAR-D stories ready (leadership, failure, conflict, technical, impact)
- [ ] Each story tested against 4 follow-up questions (prompt #4)
- [ ] Prepared a small work sample relevant to this role

## Human voice check
- [ ] Cover note / messages rewritten in my own voice (prompt #5)
- [ ] No generic AI phrasing that triggers auto-dismissal

## Follow-up
- [ ] Response received? (Y/N) __________
- [ ] What to change next time: __________
